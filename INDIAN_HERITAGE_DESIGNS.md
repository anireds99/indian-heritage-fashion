## Design Specifications

### Color Palette (Indian Heritage Collection)
- **Saffron Orange:** #FF9933 (Indian flag, temple marigolds)
- **India Green:** #138808 (Indian flag, nature)
- **Navy Blue:** #000080 (ISRO, deep tradition)
- **Sandstone Beige:** #C2B280 (ancient temples)
- **Gold Zari:** #FFD700 (royal embroidery)
- **Terracotta Red:** #E2725B (traditional pottery)
- **Ivory White:** #FFFFF0 (marble temples)

### Typography
- **Primary:** Noto Sans Devanagari (for Hindi text)
- **Secondary:** Playfair Display (for English titles)
- **Accent:** Courier (for coordinates, dates)

### Print Techniques
1. **Screen Printing** - Temple designs, architectural drawings
2. **Embroidery** - Zari work, logos, detailed motifs
3. **Hand Painting** - Scarf mandalas (Pattachitra art)
4. **Digital Print** - Color blocking, complex patterns
5. **AppliquÃ©** - Layered fabric patches

### Sustainability Features
- Organic cotton certified by GOTS
- Natural dyes (indigo, turmeric, madder root)
- Upcycled fabrics for patchwork elements
- Biodegradable packaging with seed paper tags
- Carbon-neutral shipping
